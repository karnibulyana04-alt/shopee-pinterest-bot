Privacy Policy

Last updated: July 2026

This application ("the App") is a personal-use tool that connects to the Pinterest API to create and schedule Pins on behalf of its owner's own Pinterest account.

What data we access

The App accesses the following data via the Pinterest API, only for the account that explicitly authorizes it via OAuth:

  • Pinterest board list (to select where Pins are published)
  • Pin creation data (images, titles, descriptions, links) submitted by the App owner

How data is used

Data is used solely to create and schedule Pins on the authorizing user's own Pinterest account. No data is sold, shared, or used for advertising purposes.

Data storage

Access tokens are stored locally on the App's private server and are not shared with any third party. No Pinterest user content is stored beyond what is necessary to operate the scheduling feature.

Third-party services

The App uses the Anthropic API to generate text captions. No personal Pinterest data is sent to this service — only product titles and descriptions are used to generate captions.

Contact

For questions about this policy, please contact the app owner via the GitHub repository associated with this project.
